Lab 6: Favicon, Banner & Logo


What exactly is a favicon?

A favicon, short for favorite icon, is a file that consists of an image that represents a website. It is the small icon that is located in the address bar or tab of a browser.

How are favicons created?

Favicons are created using any image editors. It uses file extensions such as .ico, .png, or .gif.

How did I create the favicon for this site?

I created the favicon for this site using a width and height of 19 bytes. It also has 6 bits per pixel. I use two different colors: a salmon color (110101) and white (111111).
